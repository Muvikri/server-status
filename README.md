# Server Status

A simple and clean Node.js based web interface for your home server! This tool includes:
- **CPU** and **Temperature** monitoring
- **RAM** usage percentage monitoring
- **Disk** usage monitoring
- **Network** usage monitoring
- **Server Uptime** monitoring

# Disclaimer
This tool is only tested in Ubuntu 24.04. I have tested it in Windows and it has many issues. Please consider using an Ubuntu server for this tool to work well.
